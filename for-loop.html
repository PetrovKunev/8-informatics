01

Какво е цикъл?

🔄 Определение

Цикълът е структура, която повторява блок код определен брой пъти. Вместо да напишеш:

Console.WriteLine("Привет");
Console.WriteLine("Привет");
Console.WriteLine("Привет");
Console.WriteLine("Привет");
Console.WriteLine("Привет"); // 100 пъти! 😱

...можеш просто да напишеш:

for (int i = 0; i < 100; i++)
{
    Console.WriteLine("Привет");
}

✅ Кога ги ползваме?

  • Да отпечатаме числа от 1 до 1000
  • Да сумираме числа
  • Да обработваме елементи на масив
  • Да рисуваме шаблони (триъгълник от звездички, таблица...)
  • Да четем редове от файл
  • Всяко "повторяй X пъти" задание

Синтаксис на for цикъл

C# Общ синтаксис
for ( тип i = начало; условие; стъпка ) { // Това се повтаря }

⚙️ Четирите части на for цикъла

  • Инициализация (i = 0) — начална стойност на брояча
  • Условие (i < 10) — "повтарай докато това е true"
  • Тяло { ... } — кодът, който се повтаря
  • Стъпка (i++) — промяната на брояча след всяка итерация

Провери условието
⬇ да
3. Изпълни тялото
Кодът вътре в { }
⬇
4. i++
Увеличи брояча
↻ назад към 2
Условието е false → СТОП

⚡ Мнемотехника: INIT → CHECK → DO → STEP

INIT: Начало — int i = 0
CHECK: Проверка — i < 10 е li true?
DO: Действие — изпълни блока вътре в { }
STEP: Стъпка — i++, после отново CHECK

Как работи стъпка по стъпка

C# Пример: числа от 1 до 5
for (int i = 1; i <= 5; i++) { Console.WriteLine(i); }

Какво се случва всяка итерация:

Итерация i стойност Условие (i <= 5) Действие След това
Начало i = 1 true Отпечатай 1 i++ → i = 2
2 i = 2 true Отпечатай 2 i++ → i = 3
3 i = 3 true Отпечатай 3 i++ → i = 4
4 i = 4 true Отпечатай 4 i++ → i = 5
5 i = 5 true Отпечатай 5 i++ → i = 6
6 i = 6 false СТОП Цикълът е приключил

Примери на for цикли

Пример 1: Сума на числа от 1 до N

C#
// Суммираме: 1 + 2 + 3 + ... + 10 int sum = 0; for (int i = 1; i <= 10; i++) { sum += i; // Добави i към сумата } Console.WriteLine("Сума: " + sum); // Изход: 55

Пример 2: Четни числа

C#
// Отпечатай четни числа от 2 до 10 for (int i = 2; i <= 10; i += 2) // Стъпка е 2! { Console.WriteLine(i); } // Изход: 2, 4, 6, 8, 10

Пример 3: Таблица за умножение

C#
// Таблица за умножение на 5 int n = 5; for (int i = 1; i <= 10; i++) { Console.WriteLine(n + " x " + i + " = " + (n * i)); } // Изход: 5 x 1 = 5, 5 x 2 = 10, ... 5 x 10 = 50

Пример 4: Обратно броене

C#
// От 10 до 1 (намаляване) for (int i = 10; i >= 1; i--) // i-- намалява { Console.WriteLine(i); } // Изход: 10, 9, 8, ... 1
05

Чести грешки

Грешка 1: Off-by-one (един повече/един по-малко)

❌ Грешно

Искаме 10 числа, но получавам 11:

for (int i = 0; i <= 10; i++)

Това дава 0,1,2...10 = 11 числа

❌ Грешно

Iskane 1-10, но получавам 0-9:

for (int i = 0; i < 10; i++)

Това дава 0,1,2...9 = 10 числа, но начало е 0

✅ Правилно

За числа 1 до 10:

for (int i = 1; i <= 10; i++)

Точно 10 числа: 1,2,3...10

Грешка 2: Бесконечен цикъл

❌ Грешно

Условието е винаги true:

for (int i = 0; i < 100; i--)

i намалява (i--), но условието е i < 100 — нямо край!

❌ Грешно

Брояч не се променя:

for (int i = 0; i < 100;)

Няма i++, така че i остава 0 завинаги!

✅ Правилно

Условието и броячът се договарят:

for (int i = 0; i < 100; i++)

i расте с i++, условието е i < 100 → край е ясен

Грешка 3: Модифициране на брояча вътре в тялото

❌ Опасно

for (int i=0; i<5; i++)
{
    i += 2; // Модифициране тук!
    Console.WriteLine(i);
}

Резултат: 2, 5 (а не 0,1,2,3,4)

✅ Правилно

for (int i=0; i<10; i+=2)
{
    Console.WriteLine(i);
}

Стъпката в ++ част е по-ясна

Вложени цикли

🔗 Що е вложен цикъл?

Цикъл вътре в цикъл. Външният се повтаря N пъти, вътрешният се повтаря M пъти за всяка итерация на външния. Общо: N × M итерации.

Пример: Правоъгълник от звездички

C# 3 реда × 4 звездички на ред
// Външен цикъл = редове for (int row = 1; row <= 3; row++) { // Вътрешен цикъл = колони for (int col = 1; col <= 4; col++) { Console.Write("* "); } Console.WriteLine(); // Нов ред }

📤 Изход

* * * *
* * * *
* * * *

Пример 2: Таблица за умножение (всички)

C# Таблица 1-10 × 1-10
for (int i = 1; i <= 3; i++) // Редове { for (int j = 1; j <= 3; j++) // Колони { Console.Write((i * j) + " "); } Console.WriteLine(); }

📤 Изход (само първите 3×3)

2 4 6
3 6 9

1
лесно Числа от 1 до 20
Напишете програма, която отпечатва всички числа от 1 до 20 на отделни редове.
  • Ползвай for цикъл със начало 1, край 20
  • С WriteLine(i) отпечатай всяко число
for (int i = 1; i <= 20; i++) { Console.WriteLine(i); }
using System; class Program { static void Main() { for (int i = 1; i <= 20; i++) { Console.WriteLine(i); } } }
2
лесно Таблица за умножение
Напишете програма, която прочита число N от потребителя и отпечатва неговата таблица за умножение (от 1 до 10).
  • С int.Parse() прочети числото от конзолата
  • Использй for от 1 до 10
  • Отпечатай: N × i = резултат
int n = int.Parse(Console.ReadLine()); for (int i = 1; i <= 10; i++) { Console.WriteLine(n + " x " + i + " = " + (n * i)); }
using System; class Program { static void Main() { Console.Write("Въведи число: "); int n = int.Parse(Console.ReadLine()); for (int i = 1; i <= 10; i++) { Console.WriteLine(n + " x " + i + " = " + (n * i)); } } }
3
средно Сума на числа
Напишете програма, която суммира всички числа от 1 до N (N прочетено от потребителя).
  • Прочети N от конзолата
  • Сумирай в променлива sum = 0
  • Использй sum += i в цикъла
  • Отпечатай резултата
int n = int.Parse(Console.ReadLine()); int sum = 0; for (int i = 1; i <= n; i++) { sum += i; } Console.WriteLine("Сума: " + sum);
using System; class Program { static void Main() { Console.Write("Въведи число N: "); int n = int.Parse(Console.ReadLine()); int sum = 0; for (int i = 1; i <= n; i++) { sum += i; } Console.WriteLine("Сума от 1 до " + n + " е: " + sum); } }
4
средно Четни числа
Напишете програма, която брои колко четни числа има в диапазона от 1 до 100.
  • Использй for от 1 до 100
  • Провери: if (i % 2 == 0)
  • Увеличи брояч при четно число
  • Отпечатай броя
int count = 0; for (int i = 1; i <= 100; i++) { if (i % 2 == 0) { count++; } } Console.WriteLine("Четни числа: " + count);
using System; class Program { static void Main() { int count = 0; for (int i = 1; i <= 100; i++) { if (i % 2 == 0) { count++; } } Console.WriteLine("Брой четни числа от 1 до 100: " + count); } }
5
трудно Факториел
Напишете програма, която пресмята факториела на число N. Факториелът е N! = 1 × 2 × 3 × ... × N.
  • Прочети N от конзолата
  • Начни с factorial = 1
  • Умножи с всяко число от 1 до N
  • Отпечатай резултата
int n = int.Parse(Console.ReadLine()); int factorial = 1; for (int i = 1; i <= n; i++) { factorial *= i; } Console.WriteLine(n + "! = " + factorial);
using System; class Program { static void Main() { Console.Write("Въведи число N: "); int n = int.Parse(Console.ReadLine()); int factorial = 1; for (int i = 1; i <= n; i++) { factorial *= i; } Console.WriteLine(n + "! = " + factorial); Console.WriteLine("Тоест: 1 × 2 × 3 × ... × " + n + " = " + factorial); } }
★
бонус Триъгълник от звездички
Напишете програма, която отпечатва триъгълник. През N редове, редица i има i звездички.
  • Прочети N от потребителя
  • 외eshен цикъл за редовете (1 до N)

int n = int.Parse(Console.ReadLine()); for (int i = 1; i <= n; i++) { for (int j = 1; j <= i; j++) { Console.Write("* "); } Console.WriteLine(); }
using System; class Program { static void Main() { Console.Write("Въведи височина на триъгълника: "); int n = int.Parse(Console.ReadLine()); for (int i = 1; i <= n; i++) { for (int j = 1; j <= i; j++) { Console.Write("* "); } Console.WriteLine(); } } } /* Пример за n = 4: * * * * * * * * * * */
