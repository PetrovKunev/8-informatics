01

КАКВО Е УСЛОВЕН ОПЕРАТОР?

★ ОСНОВНА ИДЕЯ

В реалния живот всеки ден вземаме решения на основата на условия:

  • АКО светофарът е червен → спирам
  • АКО светофарът е зелен → отивам напред
  • АКО съм гладен → ям, ИНАЧЕ продължавам

Програмирането е същото. Казваме на компютъра: "АКО условието е вярно, направи това, ИНАЧЕ направи онова".

◆ ЗА КАКВО СЛУЖАТ?

  • Проверка на стойности (четно ли е число, положително ли е)
  • Вземане на различни действия според ситуацията
  • Проверка на възраст, на пароли, на резултати
  • Всичко, което зависи от някакво условие

▲ ПРИМЕР: СВЕТОФАР

Представи си светофар:

  • Червено (true) → спирам и чакам
  • Зелено (true) → отивам напред
  • Жълто (true) → подготвям се

Точно така работят условните оператори в C# — проверяват условие и действат според резултата.

⚡ А АКО УСЛОВИЕТО Е FALSE?

Ако условието е грешно (FALSE), с помощта на else казваме: "тогава направи това друго".

C#if...else

⟳ ELSE IF — МНОГО УСЛОВИЯ

Когато имаш три или повече варианта, използвай else if.

C#класификация на оценки
int оценка = 4;

if (оценка == 6)
{
    Console.WriteLine("Отличен!");
}
else if (оценка == 5)
{
    Console.WriteLine("Много добър!");

{
    Console.WriteLine("Добър");
}
else if (оценка == 3)
{
    Console.WriteLine("Среден");
}
else
{
    Console.WriteLine("Слаб резултат");
}
04

ОПЕРАТОРИ ЗА СРАВНЕНИЕ

⚙ КАКВО СА ТЕ?

Всяко условие проверява нещо. Това става с оператори за сравнение — те казват на програмата: "сравни две стойности и ми кажи дали са еднакви, различни, по-малки и т.н."

// СРАВНЕНИЕ

==
Равно
Две стойности са еднакви
5 == 5 → TRUE
!=
Не е равно
Две стойности са различни
5 != 3 → TRUE
<
По-малко
Първото е по-малко от второто
3 < 5 → TRUE
>
По-голямо
Първото е по-голямо от второто
5 > 3 → TRUE
<=
По-малко или равно
По-малко ИЛИ точно равно
5 <= 5 → TRUE
>=
По-голямо или равно
По-голямо ИЛИ точно равно
5 >= 3 → TRUE

// ЛОГИЧЕСКИ

&&
И (AND)
Двете условия да са верни
възраст>=18 && има_паспорт
||
ИЛИ (OR)
Поне едно да е вярно
ден==6 || ден==7
!
НЕ (NOT)
Обръща значението
!(възраст<18) → пълнолетен
C#логически оператори
// && означава И
if (възраст >= 18 && има_паспорт == true)
{
    Console.WriteLine("Можеш да пътуваш в чужбина!");
}

// || означава ИЛИ
if (име == "Иван" || име == "Мария")
{
    Console.WriteLine("Име от списъка");
}

// ! означава НЕ
if (!(възраст < 18))
{
    Console.WriteLine("Пълнолетен");
}
05

ПРИМЕРИ

① ЧЕТНОСТ НА ЧИСЛО

C#четно или нечетно
int число = 7;

if (число % 2 == 0)
{
    Console.WriteLine("Числото е ЧЕТНО");
}
else
{
    Console.WriteLine("Числото е НЕЧЕТНО");
}
// % е остатък от деление
// Ако число % 2 == 0, няма остатък → четно

② ВИСОКОСНА ГОДИНА

C#високосна?
int година = 2024;

if ((година % 4 == 0 && година % 100 != 0) || година % 400 == 0)
{
    Console.WriteLine(година + " е високосна!");
}
else
{
    Console.WriteLine(година + " НЕ е високосна");
}
// Правило: дели се на 4 И НЕ на 100, ИЛИ дели се на 400

③ НАМИРАНЕ НА МИНИМУМ

C#най-малкото от три числа
int a = 15, b = 8, c = 12;

if (a < b && a < c)
{
    Console.WriteLine("Минимум: " + a);
}
else if (b < a && b < c)
{
    Console.WriteLine("Минимум: " + b);
}
else
{
    Console.WriteLine("Минимум: " + c);
}
// Отпечатва: "Минимум: 8"

ВЛОЖЕНИ УСЛОВИЯ И SWITCH

▣ ВЛОЖЕНИ УСЛОВИЯ (IF В IF)

Понякога трябва да провериш условие, след това ново условие вътре в първото. Това е вложено условие.

C#вложен if
int възраст = 20;
string водачски = "има";

if (възраст >= 18)
{
    if (водачски == "има")
    {
        Console.WriteLine("Можеш да караш кола!");
    }
    else
    {
        Console.WriteLine("Трябва ти водачски");
    }
}
else
{
    Console.WriteLine("Твърде млад си");
}

⌾ SWITCH — МНОГО УСЛОВИЯ ПО ЕДНА СТОЙНОСТ

Когато имаш много условия по една и съща променлива, използвай switch.

C#switch синтаксис
switch (променлива)
{
    case стойност1:
        // код ако променлива == стойност1
        break;
    case стойност2:
        // код ако променлива == стойност2
        break;
    default:
        // ако нито един case не съвпада
        break;
}
C#дни от седмицата
int ден = 3;

switch (ден)
{
    case 1: Console.WriteLine("Понеделник"); break;
    case 2: Console.WriteLine("Вторник");    break;
    case 3: Console.WriteLine("Сряда");      break;
    case 4: Console.WriteLine("Четвъртък");  break;
    case 5: Console.WriteLine("Петък");      break;
    case 6:
    case 7: Console.WriteLine("Уикенд!");    break;
    default: Console.WriteLine("Грешен ден"); break;
}
// Отпечатва: "Сряда"

⚠ ВАЖНО: BREAK;

break; означава "излез от switch" — без него изпълнението продължава към следващия case.

08

УПРАЖНЕНИЯ

1
ЛЕСНОПоложително или отрицателно

Дадено число, провери дали е положително, отрицателно или нула.

  • число = 5
  • Ако > 0 → "Положително"
  • Ако < 0 → "Отрицателно"
  • Иначе → "Нула"
Използвай if, else if, else с операторите > и <.
int число = 5; if (число > 0) Console.WriteLine("Положително"); else if (число < 0) Console.WriteLine("Отрицателно"); else Console.WriteLine("Нула");
2
СРЕДНОМакс от три числа

Дадени три числа, намери най-голямото.

  • a = 15, b = 23, c = 11
  • Провери кое е най-голямо
  • Отпечатай максимума
Използвай && оператор: if (a > b && a > c)
int a = 15, b = 23, c = 11; if (a > b && a > c) Console.WriteLine("Макс: " + a); else if (b > a && b > c) Console.WriteLine("Макс: " + b); else Console.WriteLine("Макс: " + c);
3
ТРУДНОПроверка на парола

Проверка дали паролата е "силна" — поне 8 символа И има цифра.

  • парола = "Test1234"
  • Провери дължина >= 8
  • Провери дали съдържа цифра (0-9)
  • "Силна" или "Слаба парола"
Използвай && оператор: парола.Length >= 8 && парола.Any(char.IsDigit)
string парола = "Test1234"; if (парола.Length >= 8 && парола.Any(char.IsDigit)) Console.WriteLine("Силна парола"); else Console.WriteLine("Слаба парола");
★
БОНУСВалиден триъгълник

Дадени три страни, провери дали могат да образуват триъгълник.

  • a = 3, b = 4, c = 5
  • Правило: a+b > c && b+c > a && a+c > b
  • "Валиден" или "Невалиден"
Провери и трите неравенства с && оператор.
int a = 3, b = 4, c = 5; if ((a + b > c) && (b + c > a) && (a + c > b)) Console.WriteLine("Валиден триъгълник"); else Console.WriteLine("Невалиден триъгълник");
