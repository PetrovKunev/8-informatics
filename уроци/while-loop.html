01

КАКВО Е WHILE?

◉ ИДЕЯ

while цикъл повтаря един блок код докато условието е вярно. Щом стане невярно (false), цикълът спира. Като: "Докато съм гладен → яж. Престана ли → спри."

◈ АНАЛОГИИ ОТ ЖИВОТА

  • Докато съм буден, ще работя. Щом заспя — спирам.
  • Докато числото е по-голямо от 0, ще го деля на 2.
  • Докато има хора на опашка, продължи обслужването.
  • Докато батерията е заредена, телефонът работи.

▲ ВАЖНО

Условието се проверява ПРЕДИ всяка итерация. Ако е невярно още от началото, цикълът изобщо не се изпълнява — нито веднъж.

02

СИНТАКСИС

C#базова структура
while (условие)
{
    // тяло — повтаря се, докато условието е true
}
поток на while цикъл
Провери условие
вярно ли е?
▼ TRUE
Изпълни тяло
един кръг
↻ обратно към проверката
FALSE → ИЗХОД
цикълът спира
C#практичен пример
int i = 1;
while (i <= 5)
{
    Console.WriteLine("Число: " + i);
    i++;  // ВАЖНО: увеличаваме i
}
// цикълът спира, когато i = 6 (вече не е <= 5)

◆ КРИТИЧНО!

Трябва да променяш променливата в условието! Ако не, цикълът никога няма да спре и програмата "зависва". Това се нарича безкраен цикъл.

03

WHILE СРЕЩУ DO...WHILE

while

  • Проверка ПРЕДИ изпълнение
  • Ако условието е false → не се изпълнява нито веднъж
  • Най-често използваната форма
  • Добро за "докато x < 10"

do...while

  • Проверка СЛЕД изпълнение
  • Винаги се изпълнява поне веднъж
  • По-рядко срещано
  • Добро за менюта и повторяем вход
C#синтаксис един до друг
// ─── while: проверка ПРЕДИ ───
while (условие) {
    // тяло
}

// ─── do...while: проверка СЛЕД ───
do {
    // винаги поне веднъж
} while (условие);

◎ КОНКРЕТНА РАЗЛИКА

Сценарий: питаме потребителя за число от 1 до 10.

while: ако вече имаме валидно число → не питаме.
do...while: винаги питаме поне веднъж, после проверяваме.

04

ПРИМЕРИ ОТ РЕАЛНИЯ СВЯТ

① ИГРА — ОТГАТНИ ЧИСЛОТО

C#
int secret = 7;
int guess = 0;

while (guess != secret)
{
    Console.Write("Отгатни число: ");
    guess = int.Parse(Console.ReadLine());

    if (guess < secret) Console.WriteLine("Твърде малко!");
    else if (guess > secret) Console.WriteLine("Твърде голямо!");
}

Console.WriteLine("Браво! Числото е " + secret);

② СУМА ДОКАТО 0

C#
int sum = 0;
int number = -1;

Console.WriteLine("Въведи числа (0 = край):");

while (number != 0)
{
    Console.Write("Число: ");
    number = int.Parse(Console.ReadLine());
    sum += number;
}

Console.WriteLine("Сума: " + sum);

③ БРОЕНЕ НА ЦИФРИ В ЧИСЛО

C#
int num = 12345;
int count = 0;

while (num > 0)
{
    count++;
    num /= 10;  // премахваме последната цифра
}

Console.WriteLine("Брой цифри: " + count);  // 5

④ ВАЛИДИРАНЕ НА ВЪВЕЖДАНЕ

C#
int age = 0;

while (age < 1 || age > 120)
{
    Console.Write("Въведи възраст (1-120): ");
    age = int.Parse(Console.ReadLine());
}

Console.WriteLine("Благодаря! Възраст: " + age);
05

БЕЗКРАЕН ЦИКЪЛ · break / continue

✕ БЕЗКРАЕН ЦИКЪЛ — НЕ ПУСКАЙ!

Безкраен е цикълът, чието условие винаги е true. Програмата "зависва" и нищо не продължава.

C#пример за безкраен цикъл
// ✕ НЕ ЗАПУСКАЙ — няма изход
while (true)
{
    Console.WriteLine("Хело");
    // няма break, няма промяна → завинаги
}

▣ break — ВЕДНАГА ИЗХОД

break прекратява цикъла незабавно, независимо от условието.

C#break пример
while (true)
{
    Console.Write("Въведи 'изход' за край: ");
    string input = Console.ReadLine();

    if (input == "изход")
    {
        break;  // излизаме от цикъла
    }

    Console.WriteLine("Продължавай...");
}

▸ continue — ПРЕСКОЧИ НА СЛЕДВАЩА ИТЕРАЦИЯ

continue пропуска остатъка от тялото и минава на следващия кръг.

C#continue пример
int i = 0;
while (i < 5)
{
    i++;
    if (i == 3) continue;  // прескачаме 3
    Console.WriteLine("Число: " + i);
}
// отпечатва: 1, 2, 4, 5 (без 3)
06

WHILE ИЛИ FOR — КОГА КОЕ?

while — когато

  • Не знаеш колко пъти ще се завърти
  • Условието е сложно или зависи от вход
  • Четеш докато има данни
  • Обновяването не е редовно

for — когато

  • Знаеш точно колко пъти ще се завърти
  • Условието е просто (i < 10)
  • Обхождаш известен диапазон
  • Стъпката е редовна (i++)
C#сравнение в действие
// FOR — знам колко пъти (10)
for (int i = 0; i < 10; i++)
{
    Console.WriteLine(i);
}

// WHILE — не знам колко (зависи от потребителя)
string answer = "";
while (answer != "край")
{
    Console.Write("Команда (край = стоп): ");
    answer = Console.ReadLine();
}

08

УПРАЖНЕНИЯ

1
ЛЕСНОБрой до N

Чети число N и отпечатай числата 1, 2, ..., N с while цикъл.

  • Прочети N от потребителя
  • i = 1, while (i <= N)
  • WriteLine(i), i++
int N = int.Parse(Console.ReadLine()); int i = 1; while (i <= N) { Console.WriteLine(i); i++; }
using System; class Program { static void Main() { Console.Write("Въведи N: "); int N = int.Parse(Console.ReadLine()); int i = 1; while (i <= N) { Console.WriteLine(i); i++; } } }
2
ЛЕСНОЧетни от 2 до 20

Отпечатай всички четни числа от 2 до 20 с while цикъл.

  • i = 2, while (i <= 20)
  • WriteLine(i)
  • i += 2 (не i++)
int i = 2; while (i <= 20) { Console.WriteLine(i); i += 2; }
using System; class Program { static void Main() { int i = 2; while (i <= 20) { Console.WriteLine(i); i += 2; } } }
3
СРЕДНОСума на първите N числа

Прочети N и изчисли сумата 1 + 2 + ... + N с while цикъл.

  • Прочети N
  • sum = 0, i = 1
  • while (i <= N) { sum += i; i++; }
  • Отпечатай сумата
int N = int.Parse(Console.ReadLine()); int sum = 0, i = 1; while (i <= N) { sum += i; i++; } Console.WriteLine("Сума: " + sum);
using System; class Program { static void Main() { Console.Write("Въведи N: "); int N = int.Parse(Console.ReadLine()); int sum = 0; int i = 1; while (i <= N) { sum += i; i++; } Console.WriteLine("Сума: " + sum); } }
4
СРЕДНОПоложителни числа

Чети числа, докато потребителят не въведе отрицателно. Преброй колко са положителните.

  • number = 0, count = 0
  • while (number >= 0) — чети ново
  • Ако е положително → count++
  • Отпечатай count
int number = 0; int count = 0; while (number >= 0) { number = int.Parse(Console.ReadLine()); if (number >= 0) count++; } Console.WriteLine("Брой: " + count);
using System; class Program { static void Main() { int number = 0; int count = 0; Console.WriteLine("Числа (отрицателно = край):"); while (number >= 0) { Console.Write("Число: "); number = int.Parse(Console.ReadLine()); if (number >= 0) count++; } Console.WriteLine("Брой положителни: " + count); } }
5
ТРУДНООбърнато число

Чети число и го обърни (123 → 321) с while цикъл.

  • Чети num, reversed = 0
  • while (num > 0):
  •   digit = num % 10
  •   reversed = reversed * 10 + digit
  •   num /= 10
За 123: - last digit = 123 % 10 = 3 - премахни: 123 / 10 = 12 - reversed = reversed * 10 + digit
using System; class Program { static void Main() { Console.Write("Число: "); int num = int.Parse(Console.ReadLine()); int original = num; int reversed = 0; while (num > 0) { int digit = num % 10; reversed = reversed * 10 + digit; num /= 10; } Console.WriteLine(original + " → " + reversed); } }
★
БОНУСФакториел с while

Изчисли N! (напр. 5! = 120) с while цикъл.

  • Прочети N
  • factorial = 1, i = N
  • while (i > 0) { factorial *= i; i--; }
int N = int.Parse(Console.ReadLine()); int factorial = 1; int i = N; while (i > 0) { factorial *= i; i--; } Console.WriteLine(N + "! = " + factorial);
using System; class Program { static void Main() { Console.Write("N: "); int N = int.Parse(Console.ReadLine()); int factorial = 1; int i = N; while (i > 0) { factorial *= i; i--; } Console.WriteLine(N + "! = " + factorial); } }
