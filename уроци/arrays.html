ЗАЩО НИ ТРЯБВАТ МАСИВИ?

✕ ПРОБЛЕМ — МНОГО ПРОМЕНЛИВИ

Искаш да пазиш оценките на 5 ученика. Досега би написал:

C#стар начин — 5 променливи
int оценка1 = 5;
int оценка2 = 6;
int оценка3 = 4;
int оценка4 = 6;
int оценка5 = 5;
// а ако са 30? 1000? 😱

◉ РЕШЕНИЕ — МАСИВ

Един масив пази много стойности под едно име:

C#нов начин — един масив
int[] оценки = { 5, 6, 4, 6, 5 };
// една променлива — пет стойности ✨
02

КАКВО Е МАСИВ?

◈ ОПРЕДЕЛЕНИЕ

Масив е редица от кутийки, подредени една до друга. Всяка кутийка:

  • съдържа една стойност от един и същ тип
  • има номер — нарича се индекс

Всички кутийки са под едно общо име (името на масива).

// int[] оценки = { 5, 6, 4, 6, 5 }
5
6
4
6
5
[0]
[1]
[2]
[3]
[4]
■ стойности    ■ индекси

▲ НАЙ-ВАЖНОТО — ИНДЕКСИТЕ ЗАПОЧВАТ ОТ 0

Първият елемент е на индекс 0, не на 1. Ако масивът има 5 елемента, индексите са 0, 1, 2, 3, 4 — не 1, 2, 3, 4, 5.

СЪЗДАВАНЕ

① НАЧИН 1 — С ГОТОВИ СТОЙНОСТИ

Когато още отначало знаеш какво ще сложиш в масива.

C#
// 5 оценки, подредени директно
int[] оценки = { 5, 6, 4, 6, 5 };

// имена — масив от текстове
string[] имена = { "Ана", "Иван", "Петя" };

// числа с плаваща запетая
double[] цени = { 1.20, 2.50, 3.99 };

② НАЧИН 2 — ПРАЗЕН МАСИВ С РАЗМЕР

Когато ще пълниш стойностите по-късно — от потребителя или от файл.

C#
// масив за 10 числа — всички започват с 0
int[] числа = new int[10];

// масив за 3 текста — всички започват с null
string[] думи = new string[3];

◎ КОГА КОЙ НАЧИН?

  • Начин 1 — когато знаеш стойностите (напр. дните на седмицата)
  • Начин 2 — когато ще четеш от потребителя или файл, а само знаеш броя

◆ СВОЙСТВОТО .Length

За да разбереш колко елемента има в масива, използваш .Length:

C#

Console.WriteLine(числа[числа.Length - 1]); // 40

ОБХОЖДАНЕ С for ЦИКЪЛ

↻ МАСИВИ + FOR = НАЙ-ДОБРИТЕ ПРИЯТЕЛИ

За да обходим всички елементи, използваме for от 0 до .Length - 1:

C#отпечатване на всички елементи
string[] имена = { "Ана", "Иван", "Петя" };

for (int i = 0; i < имена.Length; i++)
{
    Console.WriteLine(имена[i]);
}
// изход: Ана, Иван, Петя

◆ ЗАБЕЛЕЖИ УСЛОВИЕТО: i < имена.Length

Пишем < (по-малко), не <=. Защото:

  • имена.Length = 3
  • валидни индекси = 0, 1, 2
  • индекс 3 не съществува → i <= 3 дава грешка!

◈ ЧЕТЕНЕ ОТ ПОТРЕБИТЕЛ

C#пълним празен масив с for
// празен масив за 5 числа
int[] числа = new int[5];

for (int i = 0; i < числа.Length; i++)
{
    Console.Write("Число " + (i + 1) + ": ");
    числа[i] = int.Parse(Console.ReadLine());
}

ЧЕСТИ ГРЕШКИ

✕ ГРЕШКА 1 — IndexOutOfRangeException

✕ ИЗВЪН ГРАНИЦИТЕ

Масив от 3 елемента, а търсим индекс 3:

int[] a = { 1, 2, 3 };
Console.WriteLine(a[3]);

💥 Грешка! Валидни са 0, 1, 2.

✕ ОТРИЦАТЕЛЕН ИНДЕКС

C# не позволява отрицателни:

int[] a = { 1, 2, 3 };
Console.WriteLine(a[-1]);

💥 Грешка! Няма индекс −1.

✓ ПРАВИЛНО

Последният = Length − 1:

int[] a = { 1, 2, 3 };
Console.WriteLine(a[a.Length-1]);

Работи за всяка дължина.

✕ ГРЕШКА 2 — <= ВМЕСТО < В ЦИКЪЛА

✕ ГРЕШНО

for (int i=0; i <= a.Length; i++)
{
    Console.WriteLine(a[i]);
}

На последната итерация i = Length → 💥 IndexOutOfRangeException

✓ ПРАВИЛНО

for (int i=0; i < a.Length; i++)
{
    Console.WriteLine(a[i]);
}

Спираме точно преди Length.

✕ ГРЕШКА 3 — РАЗМЕР vs ПОСЛЕДЕН ИНДЕКС

Масив с 10 елемента:

  • .Length е 10 (брой елементи)
  • Последен индекс е 9 (Length − 1)

Това не са едно и също число!

ЛЕСНООтпечатай масив

Създай масив с числата 10, 20, 30, 40, 50 и отпечатай всеки на нов ред.

  • Създай int[] с 5 стойности
  • for цикъл от 0 до .Length − 1
  • Console.WriteLine(числа[i])
int[] числа = { 10, 20, 30, 40, 50 }; for (int i = 0; i < числа.Length; i++) { Console.WriteLine(числа[i]); }
using System; class Program { static void Main() { int[] числа = { 10, 20, 30, 40, 50 }; for (int i = 0; i < числа.Length; i++) { Console.WriteLine(числа[i]); } } }
2
ЛЕСНОСредна оценка

Даден е масив с оценките на ученик. Изчисли и отпечатай средната оценка.

  • double[] оценки = { 5.50, 6.00, 4.75, 5.25, 6.00 };
  • Сумирай всички с for цикъл
  • Раздели на .Length
double сума = 0; for (int i = 0; i < оценки.Length; i++) { сума += оценки[i]; } double средна = сума / оценки.Length;
using System; class Program { static void Main() { double[] оценки = { 5.50, 6.00, 4.75, 5.25, 6.00 }; double сума = 0; for (int i = 0; i < оценки.Length; i++) { сума += оценки[i]; } double средна = сума / оценки.Length; Console.WriteLine("Средна оценка: " + средна); } }
3
СРЕДНОЧетене от потребителя

Прочети N числа и ги отпечатай в обратен ред.

  • Прочети N от конзолата
  • Създай int[] с размер N
  • Запиши числата в цикъл
  • Отпечатай от N − 1 до 0
int n = int.Parse(Console.ReadLine()); int[] числа = new int[n]; for (int i = 0; i < n; i++) числа[i] = int.Parse(Console.ReadLine()); for (int i = n - 1; i >= 0; i--) Console.WriteLine(числа[i]);
using System; class Program { static void Main() { Console.Write("Брой числа: "); int n = int.Parse(Console.ReadLine()); int[] числа = new int[n]; for (int i = 0; i < n; i++) { Console.Write("Число " + (i + 1) + ": "); числа[i] = int.Parse(Console.ReadLine()); } Console.WriteLine("В обратен ред:"); for (int i = n - 1; i >= 0; i--) { Console.WriteLine(числа[i]); } } }

СРЕДНОMin и max

Даден масив. Намери най-малкото и най-голямото число.

  • int[] числа = { 7, 2, 9, 4, 11, 3, 8 };
  • min = числа[0], max = числа[0]
  • Обходи от индекс 1
  • Актуализирай при нужда
int min = числа[0]; int max = числа[0]; for (int i = 1; i < числа.Length; i++) { if (числа[i] < min) min = числа[i]; if (числа[i] > max) max = числа[i]; }
using System; class Program { static void Main() { int[] числа = { 7, 2, 9, 4, 11, 3, 8 }; int min = числа[0]; int max = числа[0]; for (int i = 1; i < числа.Length; i++) { if (числа[i] < min) min = числа[i]; if (числа[i] > max) max = числа[i]; } Console.WriteLine("Най-малко: " + min); Console.WriteLine("Най-голямо: " + max); } }
