StreamReader &&
StreamWriter

Две сестри-класове от System.IO. Едната чете от файл. Другата записва във файл. В този урок ще ги използваме заедно — за да премине информация от файл → програма → нов файл.

01 ФАЙЛЪТ Е КАТО ТЕТРАДКА

Текстовият файл (.txt) съхранява информация като обикновен текст — редове, разделени с нов ред. Програмата може да го ЧЕТЕ или да ЗАПИСВА в него.

◎ АНАЛОГИЯ · ДВЕ РЪЦЕ

Програмата има дясна ръка (StreamReader) — с нея чете това, което пише в тетрадката. И лява ръка (StreamWriter) — с нея пише нови редове в нея.

Ключова идея: Потокът е еднопосочна връзка. StreamReader чете само от файл. StreamWriter пише само във файл. Не може едната да прави и двете.

02 READER vs WRITER — СРАВНЕНИЕ

Двата класа си приличат по много неща, но имат различни задачи. Ето разликите една до друга:

◈ StreamReader

  • Чете данни ОТ файл
  • Файлът трябва да съществува
  • Не може да променя файла
  • Методи: ReadLine(), ReadToEnd()
  • Има свойство EndOfStream
  • Пример: зареждане на резултати

◈ StreamWriter

  • Записва данни ВЪВ файл
  • Ако файлът липсва — създава го
  • По подразбиране изтрива старото
  • Методи: Write(), WriteLine()
  • Опция true за добавяне (append)
  • Пример: запазване на резултати

◈ ОБЩОТО МЕЖДУ ТЯХ

⚙

И двата са в System.IO — трябва using System.IO; най-горе. И двата трябва да бъдат затворени с Close() или using блок. И двата работят с текстови файлове ред по ред.

03 StreamReader — ЧЕТЕНЕ ОТ ФАЙЛ

◈ ОСНОВЕН СИНТАКСИС · 4 СТЪПКИ

Program.csREADER
// 1. Добавяме namespace
using System.IO;

// 2. Създаваме обект — посочваме файла
StreamReader reader = new StreamReader("students.txt");

// 3. Четем ред по ред докато файлът не свърши
while (!reader.EndOfStream)
{
    string line = reader.ReadLine();
    Console.WriteLine(line);
}

// 4. Затваряме файла — ЗАДЪЛЖИТЕЛНО
reader.Close();

05 StreamWriter — ЗАПИС ВЪВ ФАЙЛ

Сега преминаваме към StreamWriter. Има два режима — важно е да знаеш разликата:

◈ РЕЖИМ · ПРЕЗАПИС

new StreamWriter("file.txt")

Ако файлът съществува — старото съдържание се изтрива. Като нов лист хартия.

◈ РЕЖИМ · APPEND

new StreamWriter("file.txt", true)

Добавя в края на файла. Старото се запазва.

◈ ОСНОВЕН СИНТАКСИС

Program.csWRITER
// Режим 1: ПРЕЗАПИС (изтрива старото съдържание)
StreamWriter writer = new StreamWriter("output.txt");

writer.WriteLine("Иван Петров");     // ред + нов ред
writer.Write("Без нов ред");          // без \n накрая
writer.Close();

// ───────────────────────────────────────────────

// Режим 2: APPEND (добавяне, без изтриване)
StreamWriter w2 = new StreamWriter("output.txt", true);
w2.WriteLine("Нов ред в края");
w2.Close();
1

ОТВОРИ

new StreamWriter( "file.txt")
2

ПИШИ

w.WriteLine("..."); w.Write("...");
3

ЗАТВОРИ

w.Close(); // flush + save!

07 МЕТОДИ И СВОЙСТВА

◈ STREAMREADER

ReadLine()
→ string | null
Чете един ред. При края на файла връща null.
ReadToEnd()
→ string
Чете целия оставащ текст наведнъж като един string.
EndOfStream
свойство · bool
Връща true когато сме стигнали края на файла.
Close()
→ void
Затваря файла и освобождава ресурсите. Задължително!

◈ STREAMWRITER

WriteLine(текст)
→ void
Записва ред и добавя нов ред накрая (като Enter).
Write(текст)
→ void
Записва текст без нов ред — продължава на същия ред.
Flush()
→ void
Принудително записва буфера. Обикн. Close() го прави автоматично.
Close()
→ void
Записва буфера и затваря файла. Без него данните може да не се запазят!
▲

Честа грешка: Ако забравиш Close() на StreamWriter, последните редове може да не бъдат записани — те са в паметта (буфера), но не са стигнали до файла!

08 ПРИМЕРНИ ПРОГРАМИ

Ще работим с файл scores.txt:

scores.txtDATA
Иван 85
Мария 92
Стефан 67
Ана 78
Петър 55

◈ ПРИМЕР 1 · Прочитане и извеждане

Example1.csREADER
using System;
using System.IO;

class Program
{
    static void Main()
    {
        StreamReader reader = new StreamReader("scores.txt");

        while (!reader.EndOfStream)
        {
            string line = reader.ReadLine();
            Console.WriteLine(line);
        }
        reader.Close();
    }
}

◈ ПРИМЕР 2 · Намери средния резултат

Example2.csREADER
StreamReader reader = new StreamReader("scores.txt");
int sum = 0, count = 0;

while (!reader.EndOfStream)
{
    string line = reader.ReadLine();     // "Иван 85"
    string[] parts = line.Split(' ');      // ["Иван","85"]
    int score = int.Parse(parts[1]);         // 85

    sum += score;
    count++;
}
reader.Close();

double avg = (double)sum / count;
Console.WriteLine($"Среден резултат: {avg:F2}");

◈ ПРИМЕР 3 · Запис от конзолата

Example3.csWRITER
StreamWriter writer = new StreamWriter("names.txt");
Console.WriteLine("Въведи 3 имена:");

for (int i = 1; i <= 3; i++)
{
    Console.Write($"Ученик {i}: ");
    string name = Console.ReadLine();
    writer.WriteLine(name);
}
writer.Close();
Console.WriteLine("Имената са записани в names.txt");

◈ ПРИМЕР 4 · READER + WRITER заедно (филтриране)

Прочети scores.txt → запиши само отличниците (≥80) в top.txt:

Example4.csREAD + WRITE
StreamReader reader = new StreamReader("scores.txt");
StreamWriter writer = new StreamWriter("top.txt");

while (!reader.EndOfStream)
{
    string line = reader.ReadLine();
    string[] parts = line.Split(' ');
    int score = int.Parse(parts[1]);

    if (score >= 80)
    {
        writer.WriteLine(line);       // само >= 80
    }
}
reader.Close();
writer.Close();
Console.WriteLine("Отличниците са в top.txt");

◈ ПРИМЕР 5 · Append към лог файл

Example5.csAPPEND
// true = добавяме в края, не изтриваме старото
StreamWriter writer = new StreamWriter("log.txt", true);
writer.WriteLine("Програмата стартира отново");
writer.Close();

// При всяко стартиране се добавя нов ред,
// без да се изтрива предишното съдържание!

09 using БЛОК — БЕЗОПАСНИЯТ НАЧИН

Какво става, ако програмата гръмне с грешка преди да стигне до Close()? Файлът остава заключен. Лошо.

◎ АНАЛОГИЯ · КРАН С ТАЙМЕР

Блокът using е като кран, който сам се затваря след употреба. Не можеш да забравиш да го затвориш — системата се грижи за теб. Работи дори при грешка.

✖ БЕЗ using (РИСКОВАНО)
StreamReader r = new StreamReader("f.txt");

// ... код ...

r.Close();
Ако стане грешка преди Close() → файлът остава заключен!
✓ С using (ПРАВИЛНО)
using (StreamReader r = new StreamReader("f.txt"))
{
    // ... код ...
}
// Close() автоматично!
Дори при грешка — Close() винаги се вика.

◈ using · READER И WRITER

Program.csusing
// StreamReader с using
using (StreamReader reader = new StreamReader("scores.txt"))
{
    while (!reader.EndOfStream)
    {
        Console.WriteLine(reader.ReadLine());
    }
} // ← Close() автоматично

// StreamWriter с using
using (StreamWriter writer = new StreamWriter("output.txt"))
{
    writer.WriteLine("Ред 1");
    writer.WriteLine("Ред 2");
} // ← Close() + Flush() автоматично
✓

Добра практика: Използвай using блока винаги, когато е възможно. Това е стандартен начин в C# за работа с ресурси.

10 УПРАЖНЕНИЯ

Време е за практика. Всяко упражнение има подсказка и пълно решение. Опитай първо сам!

✪

ЛЕСНО БРОЙ РЕДОВЕ

Файлът names.txt съдържа имена — по едно на ред. Напиши програма, която брои колко имена има.

  • Отвори names.txt с StreamReader чрез пълен път
  • С while (!reader.EndOfStream) чети ред по ред
  • Увеличавай брояч count при всеки ред
  • Отпечатай: Брой имена: 5
int count = 0; StreamReader r = new StreamReader(@"C:\Users\...\names.txt"); while (!r.EndOfStream) { r.ReadLine(); count++; } r.Close();
using System; using System.IO; class Program { static void Main() { string path = @"C:\Users\YourName\Desktop\names.txt"; int count = 0; StreamReader reader = new StreamReader(path); while (!reader.EndOfStream) { reader.ReadLine(); count++; } reader.Close(); Console.WriteLine("Брой имена: " + count); } }
ЛЕСНО ТАБЛИЦА ЗА УМНОЖЕНИЕ

Запиши таблицата за умножение на число N (въведено от потребителя) във файл table.txt.

  • Прочети числото N от конзолата
  • Създай StreamWriter за table.txt с пълен път
  • С for цикъл от 1 до 10 запиши: N x i = резултат
  • Затвори файла и потвърди на екрана
int n = int.Parse(Console.ReadLine()); StreamWriter w = new StreamWriter(@"C:\Users\...\table.txt"); for (int i = 1; i <= 10; i++) w.WriteLine(n + " x " + i + " = " + (n * i)); w.Close();
using System; using System.IO; class Program { static void Main() { string path = @"C:\Users\YourName\Desktop\table.txt"; Console.Write("Въведи число: "); int n = int.Parse(Console.ReadLine()); StreamWriter writer = new StreamWriter(path); for (int i = 1; i <= 10; i++) { writer.WriteLine(n + " x " + i + " = " + (n * i)); } writer.Close(); Console.WriteLine("Таблицата е записана в: " + path); } }
СРЕДНО МАКСИМАЛЕН РЕЗУЛТАТ

Файлът numbers.txt съдържа по едно цяло число на ред. Намери и отпечатай най-голямото число.

  • Прочети първия ред и го задай като начален максимум
  • Чети останалите с while
  • Ако текущото число > max → обнови max
  • Отпечатай Максимум: X
StreamReader r = new StreamReader(@"C:\Users\...\numbers.txt"); int max = int.Parse(r.ReadLine()); while (!r.EndOfStream) { int n = int.Parse(r.ReadLine()); if (n > max) max = n; } r.Close();
using System; using System.IO; class Program { static void Main() { string path = @"C:\Users\YourName\Desktop\numbers.txt"; StreamReader reader = new StreamReader(path); int max = int.Parse(reader.ReadLine()); while (!reader.EndOfStream) { int num = int.Parse(reader.ReadLine()); if (num > max) max = num; } reader.Close(); Console.WriteLine("Максимум: " + max); } }
СРЕДНО ЧЕТНИ ЧИСЛА ВЪВ ФАЙЛ

Прочети числа от input.txt и запиши само четните в even.txt.

  • Отвори StreamReader за input.txt и StreamWriter за even.txt
  • Чети ред по ред, конвертирай с int.Parse()
  • Ако n % 2 == 0 → запиши в even.txt
  • Затвори и двата потока
StreamReader r = new StreamReader(@"C:\Users\...\input.txt"); StreamWriter w = new StreamWriter(@"C:\Users\...\even.txt"); while (!r.EndOfStream) { int n = int.Parse(r.ReadLine()); if (n % 2 == 0) w.WriteLine(n); } r.Close(); w.Close();
using System; using System.IO; class Program { static void Main() { string inputPath = @"C:\Users\YourName\Desktop\input.txt"; string outputPath = @"C:\Users\YourName\Desktop\even.txt"; StreamReader reader = new StreamReader(inputPath); StreamWriter writer = new StreamWriter(outputPath); while (!reader.EndOfStream) { int num = int.Parse(reader.ReadLine()); if (num % 2 == 0) writer.WriteLine(num); } reader.Close(); writer.Close(); Console.WriteLine("Четните числа са записани в: " + outputPath); } }
ТРУДНО СТАТИСТИКА И ЗАПИС

Файлът grades.txt съдържа само числа 3–6. Намери: брой, сума, средна оценка, брой слаби (<4). Запиши в stats.txt.

  • Отвори StreamReader + StreamWriter с пълни пътища
  • Натрупвай sum, count, weak
  • Запиши статистиката форматирано в stats.txt
  • Потвърди на екрана с Console.WriteLine
StreamReader r = new StreamReader(@"C:\Users\...\grades.txt"); StreamWriter w = new StreamWriter(@"C:\Users\...\stats.txt"); int sum=0, count=0, weak=0; while (!r.EndOfStream) { int g = int.Parse(r.ReadLine()); sum += g; count++; if (g < 4) weak++; } r.Close(); w.WriteLine("Брой: " + count); w.WriteLine($"Средна: {(double)sum/count:F2}"); w.WriteLine("Слаби: " + weak); w.Close();
using System; using System.IO; class Program { static void Main() { string gradesPath = @"C:\Users\YourName\Desktop\grades.txt"; string statsPath = @"C:\Users\YourName\Desktop\stats.txt"; StreamReader reader = new StreamReader(gradesPath); int sum = 0, count = 0, weak = 0; while (!reader.EndOfStream) { int grade = int.Parse(reader.ReadLine()); sum += grade; count++; if (grade < 4) weak++; } reader.Close(); StreamWriter writer = new StreamWriter(statsPath); writer.WriteLine("Брой оценки: " + count); writer.WriteLine("Сума: " + sum); writer.WriteLine($"Средна оценка: {(double)sum / count:F2}"); writer.WriteLine("Брой слаби (под 4): " + weak); writer.Close(); Console.WriteLine("Статистиката е записана в: " + statsPath); } }
БОНУС ДНЕВНИК (APPEND)

Програма-дневник. При всяко стартиране потребителят въвежда бележка, която се добавя в края на diary.txt (без да изтрива предишните).

  • Използвай StreamWriter с true (append)
  • Поискай бележката от потребителя
  • Запиши: [Запис #N]: бележката
  • Бонус: Прочети с StreamReader колко записа вече има
string path = @"C:\Users\...\diary.txt"; int count = 0; if (File.Exists(path)) { StreamReader r = new StreamReader(path); while (!r.EndOfStream) { r.ReadLine(); count++; } r.Close(); } StreamWriter w = new StreamWriter(path, true); Console.Write("Бележка: "); string note = Console.ReadLine(); w.WriteLine($"[Запис #{count+1}]: " + note); w.Close();
using System; using System.IO; class Program { static void Main() { string path = @"C:\Users\YourName\Desktop\diary.txt"; int count = 0; if (File.Exists(path)) { StreamReader reader = new StreamReader(path); while (!reader.EndOfStream) { reader.ReadLine(); count++; } reader.Close(); } Console.WriteLine("Дневникът има " + count + " записа."); Console.Write("Нова бележка: "); string note = Console.ReadLine(); StreamWriter writer = new StreamWriter(path, true); writer.WriteLine($"[Запис #{count + 1}]: {note}"); writer.Close(); Console.WriteLine("Записано в: " + path); } }
