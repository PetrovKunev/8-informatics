01

Какво е while цикъл?

💡 Идея

while цикъл повтаря един блок код докато условието е вярно. Щом условието стане невярно (false), цикълът спира. Это като да говориш: "Докато съм гладен, ще ям. Щом престана да съм гладен, спирам."

🎯 Аналогия

  • Докато съм будна, ще работя. Щом заспя — спирам.
  • Докато числото е различно от 0, ще го делям на 2.
  • Докато хората чакат в опашката, продължаваме обслужването.
  • Докато батерията има енергия, телефонът работи.

⚠️ Важно

Условието се проверя ПРЕДИ всяка итерация. Ако условието е невярно от началото, цикълът вообще не се изпълнява.

Синтаксис на while цикъл

while (условие) { // тяло на цикъла — този код се повтаря // докато условието е вярно }
// ПОТОК НА WHILE ЦИКЪЛ

C# Практичен пример
1 int i = 1; 2 while (i <= 5) 3 { 4 Console.WriteLine("Число: " + i); // Отпечата: Число: 1, 2, 3, 4, 5 5 i++; // ВАЖНО: увели i 6 } 7 // Цикълът спира, защото i е 6, което не е <= 5

🔴 Критично!

Трябва да změníш променливата в цикъл! Ако не го направиш, цикълът никога няма да спре и програмата ще "зависне". Това се нарича безкраен цикъл.

03

while срещу do...while

while

  • Проверка условие ПРЕДИ изпълнение
  • Ако условие е false → не се изпълнява вообще
  • Най-честа форма на цикъл
  • Подходящ за условия като "докато x < 10"

do...while

  • Проверка условие СЛЕД изпълнение
  • Винаги се изпълнява най-малко веднъж
  • Рядко се използва
  • Подходящ за меню или повторяемо въвеждане
C# while vs do...while
// ─── while: проверка ПРЕДИ ─── while (условие) { // тяло } // ─── do...while: проверка СЛЕД ─── do { // тяло — ВСЕГДА се изпълнява поне веднъж } while (условие);

📊 Конкретна разлика — пример

Сценарий: Прочитаме число от потребителя, което трябва да е 1 и 10.

while версия: Ако потребителят не въведе нищо, цикълът не се изпълнява.

do...while версия: Винаги ще попитаме поне веднъж, дори потребителят да не направи нищо.

04

Примери от реален свят

C# Пример 1: Угадване число (угаждане)
int secret = 7; int guess = 0; while (guess != secret) { Console.Write("Угадай число: "); guess = int.Parse(Console.ReadLine()); if (guess < secret) Console.WriteLine("Твърде малко!"); else if (guess > secret) Console.WriteLine("Твърде голямо!"); } Console.WriteLine("Вярно! Числото е " + secret);
C# Пример 2: Сума докато 0
int sum = 0; int number = 0; Console.WriteLine("Въведи числа (0 = край):"); while (number != 0) { Console.Write("Число: "); number = int.Parse(Console.ReadLine()); sum += number; } Console.WriteLine("Сума: " + sum);
C# Пример 3: Броене цифри в число
int num = 12345; int count = 0; while (num > 0) { count++; num /= 10; // премахни последната цифра } Console.WriteLine("Брой цифри: " + count); // Отпечата: 5
C# Пример 4: Валидиране на входни данни
int age = 0; Console.WriteLine("Въведи възраст (1-120):"); while (age < 1 || age > 120) { Console.Write("Невалидна възраст. Повтори: "); age = int.Parse(Console.ReadLine()); } Console.WriteLine("Благодаря! Твоята възраст е " + age);
05

Безкраен цикъл и управление

🔥 Безкраен цикъл

Безкраен цикъл е такъв, който никога не спира, защото условието винаги е истинско. Програмата "зависва" и нищо не се случва.

C# Пример на безкраен цикъл (НЕ ПУСКАЙ!)
// ❌ БЕЗКРАЕН ЦИКЪЛ — не работи! while (true) { Console.WriteLine("Хело"); // i++ няма! Цикълът ще пише "Хело" завинаги! }

🎛️ break — Излез от цикъл

break веднага прекратява цикъла, независимо дали условието е вярно.

C# break пример
while (true) { Console.Write("Въведи 'выход' за край: "); string input = Console.ReadLine(); if (input == "выход") { break; // ИЗХОД ОТ ЦИКЪЛА } Console.WriteLine("Продължаваш..."); }

⏭️ continue — Пропусни на следващ кръг

continue пропуска остатъка от кода в текущата итерация и преминава на следващия кръг на цикъла.

C# continue пример
int i = 0; while (i < 5) { i++; if (i == 3) { continue; // Пропусни числото 3 } Console.WriteLine("Число: " + i); } // Отпечата: 1 2 4 5 (без 3!)
06

while vs for — кога кое?

while

  • Не знаеш колко кръга ще имаш
  • Условие е сложно или непредсказуемо
  • Входни данни от потребител
  • Обновление не е редовно (i += различни стойности)

for

  • Знаеш точно колко кръга ще имаш
  • Условие е просто (i < 10)
  • Циклиране над известен брой
  • Редовно обновление (i++)
C# Сравнение в действие
// ─── FOR: Циклирам 10 пъти (ЗНАМ броя) ─── for (int i = 0; i < 10; i++) { Console.WriteLine(i); } // ─── WHILE: Циклирам докато потребител каже "край" (НЕ ЗНАМ броя) ─── string answer = ""; while (answer != "край") { Console.Write("Въведи команда (край за излаз): "); answer = Console.ReadLine(); }

1
лесно Брой до N
Напиши програма, която чете число N и отпечатва всички числа от 1 до N с while цикъл.
  • Попроси потребителя да въведе число N
  • Използвай while цикъл за отпечатване на 1, 2, 3, ..., N
  • Всяко число на нов ред
Трябва ти: - Променлива i = 1 - while (i <= N) - println(i) - i++
using System; class Program { static void Main() { Console.Write("Въведи N: "); int N = int.Parse(Console.ReadLine()); int i = 1; while (i <= N) { Console.WriteLine(i); i++; } } }
2
лесно Четни числа
Напиши програма, която отпечатва всички четни числа от 2 до 20 с while цикъл.
  • Начало: i = 2
  • while цикъл: i <= 20
  • Отпечатай i
  • Увеличение: i += 2 (за четни числа)
i = 2; while (i <= 20) { Console.WriteLine(i); i += 2; // добавяй 2, не 1 }
using System; class Program { static void Main() { int i = 2; while (i <= 20) { Console.WriteLine(i); i += 2; } } }
3
средно Сума N първи числа
Напиши програма, която пита потребителя за N и изчислява сумата на първите N числа (1+2+3+...+N) с while цикъл.
  • Попроси число N
  • Инициализирай sum = 0 и i = 1
  • while цикъл до N
  • sum += i
  • Отпечатай крайната сума
Трябват две променливи: - sum (за съхраняване на сума) - i (за брояч) Во while цикълa: - sum += i; - i++;
using System; class Program { static void Main() { Console.Write("Въведи N: "); int N = int.Parse(Console.ReadLine()); int sum = 0; int i = 1; while (i <= N) { sum += i; i++; } Console.WriteLine("Сума: " + sum); } }
4
средно Въвеждане на положителни числа
Напиши програма, която чита числа от потребителя докато не въведе отрицателно число. На край отпечатай броя на въведените положителни числа.
  • Инициализирай number = 0 и count = 0
  • while (number >= 0)
  • Попроси число
  • Ако е положително, увеличи count
  • Отпечатай броя
int number = 0; int count = 0; while (number >= 0) { number = int.Parse(Console.ReadLine()); if (number >= 0) count++; }
using System; class Program { static void Main() { int number = 0; int count = 0; Console.WriteLine("Въвеждай числа (отрицателно число за край):"); while (number >= 0) { Console.Write("Число: "); number = int.Parse(Console.ReadLine()); if (number >= 0) count++; } Console.WriteLine("Брой положителни: " + count); } }
5
трудно Обратна броя
Напиши програма, която чете число и го обръща (напр. 123 → 321) като използваш while цикъл. Трябва да работи с многоцифрени числа.
  • Попроси число
  • Инициализирай reversed = 0
  • while (число > 0)
  • Вземи последната цифра: число % 10
  • Добави към reversed
  • Премахни последната цифра: число /= 10
За числото 123: - последна цифра: 123 % 10 = 3 - премахни я: 123 / 10 = 12 reversed = reversed * 10 + lastDigit
using System; class Program { static void Main() { Console.Write("Въведи число: "); int num = int.Parse(Console.ReadLine()); int reversed = 0; int original = num; while (num > 0) { int digit = num % 10; reversed = reversed * 10 + digit; num /= 10; } Console.WriteLine(original + " -> " + reversed); } }
6
бонус Факториел
Напиши програма, която изчислява факториела на число (5! = 5*4*3*2*1 = 120) като използваш while цикъл. Провери с N от 1 до 5.
  • Попроси число N
  • Инициализирай factorial = 1 и i = N
  • while (i > 0): factorial *= i; i--;
  • Отпечатай факториела
  • Проверка: 5! = 120
Факториел = N * (N-1) * (N-2) * ... * 1 int factorial = 1; int i = N; while (i > 0) { factorial *= i; i--; }
using System; class Program { static void Main() { Console.Write("Въведи число N: "); int N = int.Parse(Console.ReadLine()); int factorial = 1; int i = N; while (i > 0) { factorial *= i; i--; } Console.WriteLine(N + "! = " + factorial); } }
