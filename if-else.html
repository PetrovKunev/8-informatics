01

Какво е условен оператор?

💡 Основна идея

В реалния живот всеки ден вземаме решения на основата на условия:

  • АКО светофарът е червен → спирам
  • АКО светофарът е зелен → отивам напред
  • АКО оценката е под 2 → печаля, ИНАЧЕ съм доволен

Програмирането е точно същото! Казваме на компютъра: "АКО условието е вярно, направи това, ИНАЧЕ направи онова".

🎯 За какво служат?

  • Проверяване на стойности (четно ли е число, положително ли е)
  • Вземане на различни действия според ситуацията
  • Проверка на възрастта, на пароли, на резултати
  • Всичко което зависи от някакво условие!

🚦 Пример: Светофар

Представи си светофар:

  • Червено светло (true) → Спирам и чакам
  • Зелено светло (true) → Отивам напред
  • Жълто светло (true) → Подготвям се

Точно така работят условните оператори в C# — проверяват условие и действат според резултата!

02

Синтаксис на if

📝 Основен синтаксис

Всяка проверка в C# има следния шаблон:

C# Обикновен if
if (условие) { // код, който се изпълнява АКО условието е TRUE }

❓ Как се чете?

if = АКО (проверяем условието)
(условие) = вътрешността на скобите трябва да е TRUE или FALSE
{ ... } = ако е TRUE, изпълни всичко в скобите

int възраст = 16; if (възраст >= 18) { Console.WriteLine("Дойде на 18 години, много добре!"); } // Ако възраст НЯМА 18, нищо не се случва

if...else и else if

⚡ Еска — что се случва когда условието е FALSE?

А ако е условието е грешно (FALSE)? С помощта на else казваме: "Ако първото условие е грешно, тогава направи ТОА!"

C# if...else синтаксис
if (условие) { // код АКО условието е TRUE } else { // код АКО условието е FALSE }

C# Пример с else
int възраст = 15; if (възраст >= 18) { Console.WriteLine("Можеш да управляваш кола!"); } else { Console.WriteLine("Много млад си още, почакай!"); } // Отпечатва: "Много млад си още, почакай!"

🔄 else if — множество условия

А ако имаш три или повече варианта? Използвай else if!

C# if...else if...else синтаксис
if (условие1) { // ако условие1 е TRUE } else if (условие2) { // ако условие1 е FALSE И условие2 е TRUE } else if (условие3) { // ако условие1 И условие2 са FALSE, И условие3 е TRUE } else { // ако всички са FALSE }
C# Класификация на оценки
int оценка = 4; if (оценка == 6) { Console.WriteLine("Отличен!"); } else if (оценка == 5) { Console.WriteLine("Много добър!"); } else if (оценка == 4) { Console.WriteLine("Добър"); } else if (оценка == 3) { Console.WriteLine("Среден"); } else { Console.WriteLine("Слаб резултат"); }
04

Оператори за сравнение

🔍 Какво са оператори за сравнение?

Всяко условие има нещо, което проверяваме. Това се прави с оператори за сравнение. Те казват на програмата: "Сравни две стойности и каза ми дали са еднакви, различни, по-малки и т.н."

Оператори за сравнение

==
Равно
Проверява дали две стойности са еднакви
5 == 5 → TRUE
!=
Не е равно
Проверява дали две стойности са различни
5 != 3 → TRUE
<
По-малко
Проверява дали първото е по-малко от второто
3 < 5 → TRUE
>
По-голямо
Проверява дали първото е по-голямо от второто
5 > 3 → TRUE
<=
По-малко или равно
По-малко ИЛИ точно равно
5 <= 5 → TRUE
>=
По-голямо или равно
По-голямо ИЛИ точно равно
5 >= 3 → TRUE

Логически оператори

&&
И (AND)
Двете условия трябва да са верни
(възраст >= 18) && (име != "")
||
ИЛИ (OR)
Поне едно условие трябва да е вярно
(дата == Събота) || (дата == Неделя)
!
НЕ (NOT)
Обрача истинската стойност (TRUE станало FALSE)
!(възраст < 18) → дорастнал
C# Използване на логически оператори
// && означава И (ДВЕ условия) if (възраст >= 18 && има_паспорт == true) { Console.WriteLine("Можеш да пътуваш в чужбина!"); } // || означава ИЛИ (ПОНЕ ЕДНО условие) if (име == "Иван" || име == "Мария") { Console.WriteLine("Вече имам това име!"); } // ! означава НЕ (обръща значението) if (!(възраст < 18)) { Console.WriteLine("Дорастнал си"); }
05

Примери

Пример 1️⃣ : Четност на число

C# Проверка дали число е четно или нечетно
int число = 7; if (число % 2 == 0) { Console.WriteLine("Числото е ЧЕТНО"); } else { Console.WriteLine("Числото е НЕЧЕТНО"); } // % е модулна операция (остатък от деление) // Ако число % 2 == 0, значи няма остатък → четно

Пример 2️⃣ : Класификация на оценка

C# От числова оценка до словна оценка
int оценка = 5; if (оценка == 6) { Console.WriteLine("Отличен! 🏆"); } else if (оценка == 5) { Console.WriteLine("Много добър! 👍"); } else if (оценка == 4) { Console.WriteLine("Добър 👌"); } else if (оценка == 3) { Console.WriteLine("Среден 😐"); } else { Console.WriteLine("Нужна ти помощ 😟"); }

Пример 3️⃣ : Проверка на високосна година

C# Е ли горната година високосна?
int година = 2024; if ((година % 4 == 0 && година % 100 != 0) || година % 400 == 0) { Console.WriteLine(година + " е високосна година!"); } else { Console.WriteLine(година + " НЕ е високосна година"); } // Правило: дели се на 4 И НЕ на 100, ИЛИ дели се на 400

Пример 4️⃣ : Намиране на минимума

C# От три числа намери най-малкото
int a = 15, b = 8, c = 12; if (a < b && a < c) { Console.WriteLine("Минимум: " + a); } else if (b < a && b < c) { Console.WriteLine("Минимум: " + b); } else { Console.WriteLine("Минимум: " + c); } // Отпечатва: "Минимум: 8"

Вложени условия и switch

🪆 Вложени условия (if вътре в if)

Понякога трябва да проверяваш условие, а след това трябва да проверяваш ново условие вътре в първото. Това се нарича вложени условия.

C# Вложено условие
int възраст = 20; string водачски = "има"; if (възраст >= 18) { // Първото условие е верно, сега проверяваме второ if (водачски == "има") { Console.WriteLine("Можеш да управляваш кола!"); } else { Console.WriteLine("Трябва водачски"); } } else { Console.WriteLine("Твърде млад си"); }

🎯 switch — много условия

Когато имаш много условия по един и същ факт, използвай switch. Той е като дърво решения!

C# switch синтаксис
switch (променлива) { case стойност1: // код ако променлива == стойност1 break; case стойност2: // код ако променлива == стойност2 break; default: // код ако НИ ЕДИН case не се съвпада break; }
C# switch пример
int ден = 3; switch (ден) { case 1: Console.WriteLine("Понеделник"); break; case 2: Console.WriteLine("Вторник"); break; case 3: Console.WriteLine("Сряда"); break; case 4: Console.WriteLine("Четвъртък"); break; case 5: Console.WriteLine("Петък"); break; case 6: case 7: Console.WriteLine("Уикенд!"); break; default: Console.WriteLine("Грешен ден"); break; } // Отпечатва: "Сряда"

⚠️ Важно: break;

break; означава "излез от switch" - без него ще се продължи проверката!

08

Упражнения

1
ЛЕСНА Проверка на старт

Напиши код, който проверява дали число е положително или отрицателно.

  • Въведи променлива число със стойност -5
  • Ако е >= 0: отпечатай "Положително"
  • Ако е < 0: отпечатай "Отрицателно"
Използвай if/else структура: if (число >= 0) → положително else → отрицателно
int число = -5; if (число >= 0) { Console.WriteLine("Положително"); } else { Console.WriteLine("Отрицателно"); }
2
ЛЕСНА Четност

Напиши код, който проверява дали число е четно или нечетно.

  • Въведи число 7
  • Ако число % 2 == 0: "Четно"
  • Иначе: "Нечетно"
Използвай % оператор: число % 2 == 0 → четно число % 2 != 0 → нечетно
int число = 7; if (число % 2 == 0) { Console.WriteLine("Четно"); } else { Console.WriteLine("Нечетно"); }
3
СРЕДНА Съвет за летни дрехи

Напиши код, който дава съвет за облекло въз основа на температура.

  • Дай променлива температура = 28
  • Ако >= 25: "Лятна облека!"
  • Ако 15-25: "Пулувер е идеален"
  • Ако < 15: "Нужен ти е палто"
Използвай else if за три варианта: - ако >= 25 - else if >= 15 (и < 25) - else < 15
int температура = 28; if (температура >= 25) { Console.WriteLine("Лятна облека!"); } else if (температура >= 15) { Console.WriteLine("Пулувер е идеален"); } else { Console.WriteLine("Нужен ти е палто"); }
4
СРЕДНА Детелинското число

Проверка дали число е едноцифрено, двуцифрено или трицифрено.

  • число = 42
  • Ако число е между 0-9: "Едноцифрено"
  • Ако между 10-99: "Двуцифрено"
  • Ако >= 100: "Трицифрено"
Използвай сравнения: - ако < 10 → едноцифрено - else if < 100 → двуцифрено - else → трицифрено
int число = 42; if (число < 10) { Console.WriteLine("Едноцифрено"); } else if (число < 100) { Console.WriteLine("Двуцифрено"); } else { Console.WriteLine("Трицифрено"); }
5
ТРУДНА Тест за пароля

Напиши код, който проверява дали парола е силна (най-малко 8 символа И съдържа цифра).

  • пароля = "Test1234"
  • Проверка дължина >= 8
  • Проверка съдържа ли цифра (0-9)
  • Ако двете верни: "Силна парола" иначе "Слаба парола"
Използвай && логически оператор: if (пароля.Length >= 8 && пароля.Any(char.IsDigit))
string пароля = "Test1234"; if (пароля.Length >= 8 && пароля.Any(char.IsDigit)) { Console.WriteLine("Силна парола"); } else { Console.WriteLine("Слаба парола"); } // Забележка: char.IsDigit() е вградена функция за проверка на цифра
6
БОНУС Триъгълник Valide

Дадени три страни, проверка дали могат да образуват триъгълник (сума на две страни > третата).

  • a = 3, b = 4, c = 5
  • Всички условия: a+b > c && b+c > a && a+c > b
  • Ако верни: "Валиден триъгълник", иначе: "Невалиден"
Проверка с && оператор: if ((a+b > c) && (b+c > a) && (a+c > b))
int a = 3, b = 4, c = 5; if ((a + b > c) && (b + c > a) && (a + c > b)) { Console.WriteLine("Валиден триъгълник"); } else { Console.WriteLine("Невалиден триъгълник"); }
