Какво е текстов файл?

📄 Определение

Текстовият файл (.txt) съхранява информация като обикновен текст — редове, разделени с нов ред. Програмата може да го чете (StreamReader) или да записва в него (StreamWriter) — точно като да четеш или пишеш в тетрадка.

02

StreamReader срещу StreamWriter — сравнение

📖 StreamReader

  • Чете данни от файл
  • Файлът трябва да съществува
  • Не може да променя файла
  • Използва ReadLine(), ReadToEnd()
  • Следи позицията EndOfStream
  • Пример: зареждане на резултати

✏️ StreamWriter

  • Записва данни в файл
  • Ако файлът не съществува — го създава
  • По подразбиране изтрива стария съдържание
  • Използва Write(), WriteLine()
  • Опция true за добавяне (append)
  • Пример: запазване на резултати

⚙️ Общото между тях

  • И двата са в System.IO — трябва using System.IO;
  • И двата трябва да бъдат затворени с Close() или using блок
  • И двата работят с текстови файлове ред по ред

StreamWriter — запис в файл

✏️ Два режима на StreamWriter

  • new StreamWriter("file.txt") — презаписва файла (ако има нещо — изтрива го)
  • new StreamWriter("file.txt", true) — добавя в края (append режим)
C# Основен синтаксис — StreamWriter
// Режим 1: ПРЕЗАПИСВАНЕ (изтрива старото съдържание) StreamWriter writer = new StreamWriter("output.txt"); writer.WriteLine("Иван Петров"); // записва ред + нов ред writer.Write("Без нов ред"); // записва БЕЗ нов ред накрая writer.Close(); // ───────────────────────────────────────────────── // Режим 2: ДОБАВЯНЕ (append = true) StreamWriter writer2 = new StreamWriter("output.txt", true); writer2.WriteLine("Нов ред в края"); writer2.Close();

05

Методи и свойства

StreamReader методи:

ReadLine()
→ string | null
Чете един ред. При края на файла връща null.
ReadToEnd()
→ string
Чете целия оставащ текст наведнъж като един низ.
EndOfStream
свойство: bool
Връща true когато сме стигнали края на файла.
Close()
→ void
Затваря файла и освобождава ресурсите. Винаги задължително!

StreamWriter методи:

WriteLine(текст)
→ void
Записва ред и добавя нов ред накрая (като Enter).
Write(текст)
→ void
Записва текст без нов ред накрая — продължава на същия ред.
Flush()
→ void
Принудително записва буфера в файла (обикн. Close() го прави).
Close()
→ void
Записва буфера и затваря файла. Без Close() данните може да не се запазят!

⚠️ Честа грешка

Ако забравим Close() на StreamWriter, последните редове може да не бъдат записани — те са в паметта (буфера), но не са стигнали до файла!

Примерни програми

Работим с файл scores.txt:

TXTscores.txt
Иван 85 Мария 92 Стефан 67 Ана 78 Петър 55

Пример 1 — Прочитане и извеждане на всички редове

C#StreamReader — четене ред по ред
using System; using System.IO; class Program { static void Main() { StreamReader reader = new StreamReader("scores.txt"); while (!reader.EndOfStream) { string line = reader.ReadLine(); Console.WriteLine(line); } reader.Close(); } }

Пример 2 — Намери средния резултат

C#StreamReader — обработка на данни
StreamReader reader = new StreamReader("scores.txt"); int sum = 0, count = 0; while (!reader.EndOfStream) { string line = reader.ReadLine(); // "Иван 85" string[] parts = line.Split(' '); // ["Иван", "85"] int score = int.Parse(parts[1]); // 85 sum += score; count++; } reader.Close(); double avg = (double)sum / count; Console.WriteLine($"Среден резултат: {avg:F2}");

Пример 3 — Запис на нови данни в файл

C#StreamWriter — запис от конзолата
StreamWriter writer = new StreamWriter("names.txt"); Console.WriteLine("Въведи 3 имена:"); for (int i = 1; i <= 3; i++) { Console.Write($"Ученик {i}: "); string name = Console.ReadLine(); writer.WriteLine(name); // записва в файла } writer.Close(); Console.WriteLine("Имената са записани в names.txt");

Пример 4 — Четене и запис заедно (филтриране)

C#Прочети scores.txt → запиши само отличниците в top.txt
StreamReader reader = new StreamReader("scores.txt"); StreamWriter writer = new StreamWriter("top.txt"); while (!reader.EndOfStream) { string line = reader.ReadLine(); string[] parts = line.Split(' '); int score = int.Parse(parts[1]); if (score >= 80) { writer.WriteLine(line); // записваме само >= 80 } } reader.Close(); writer.Close(); Console.WriteLine("Отличниците са в top.txt");

Пример 5 — Добавяне към съществуващ файл (append)

C#StreamWriter append — добавяне в края
// true = добавяме в края, не изтриваме старото StreamWriter writer = new StreamWriter("log.txt", true); writer.WriteLine("Програмата стартира отново"); writer.Close(); // При всяко стартиране на програмата се добавя нов ред, // без да се изтрива предишното съдържание!
07

using блок — препоръчителният начин

💡 Защо using?

Блокът using автоматично извиква Close() след края — дори ако стане грешка по средата. Не трябва да пишем Close() ръчно.

❌ Без using (рискован)

StreamReader r =
  new StreamReader("f.txt");

// ... код ...

r.Close();
// Ако стане грешка преди
// Close() → файлът
// остава заключен!

✅ С using (правилно)

using (StreamReader r =
  new StreamReader("f.txt"))
{
    // ... код ...
}
// ← Close() автоматично!
// Дори при грешка работи!
C#using блок — StreamReader и StreamWriter
// StreamReader с using using (StreamReader reader = new StreamReader("scores.txt")) { while (!reader.EndOfStream) { Console.WriteLine(reader.ReadLine()); } } // ← Close() автоматично // StreamWriter с using using (StreamWriter writer = new StreamWriter("output.txt")) { writer.WriteLine("Ред 1"); writer.WriteLine("Ред 2"); } // ← Close() + Flush() автоматично

1
лесно Брой редове
Файлът names.txt съдържа имена — по едно на ред. Напишете програма, която брои колко имена има.
Създайте файла names.txt на удобно място (напр. Desktop) и използвайте пълния път.

  • Увеличавай брояч count при всеки ред
  • Отпечатай: Брой имена: 5
int count = 0; StreamReader r = new StreamReader(@"C:\Users\...\names.txt"); // Замени пътя с реалния път до твоя файл! while (!r.EndOfStream) { r.ReadLine(); count++; } r.Close();
using System; using System.IO; class Program { static void Main() { // Запиши пълния път до файла: string path = @"C:\Users\YourName\Desktop\names.txt"; int count = 0; StreamReader reader = new StreamReader(path); while (!reader.EndOfStream) { reader.ReadLine(); count++; } reader.Close(); Console.WriteLine("Брой имена: " + count); } }
2
лесно Запис на таблица за умножение
Запишете таблицата за умножение на число N (въведено от потребителя) в файл table.txt.
Използвайте пълен път, за да знаете къде ще се създаде файлът.
  • Прочети числото N от конзолата
  • Създай StreamWriter за table.txt с пълен път
  • С for цикъл от 1 до 10 запиши: N x i = резултат

int n = int.Parse(Console.ReadLine()); StreamWriter w = new StreamWriter(@"C:\Users\...\table.txt"); for (int i = 1; i <= 10; i++) w.WriteLine(n + " x " + i + " = " + (n * i)); w.Close();
using System; using System.IO; class Program { static void Main() { string path = @"C:\Users\YourName\Desktop\table.txt"; Console.Write("Въведи число: "); int n = int.Parse(Console.ReadLine()); StreamWriter writer = new StreamWriter(path); for (int i = 1; i <= 10; i++) { writer.WriteLine(n + " x " + i + " = " + (n * i)); } writer.Close(); Console.WriteLine("Таблицата е записана в: " + path); } }

Прочети числа от input.txt и запиши само четните в even.txt.
И двата файла трябва да са с пълен път — входният и изходният.

StreamReader r = new StreamReader(@"C:\Users\...\input.txt"); StreamWriter w = new StreamWriter(@"C:\Users\...\even.txt"); while (!r.EndOfStream) { int n = int.Parse(r.ReadLine()); if (n % 2 == 0) w.WriteLine(n); } r.Close(); w.Close();
using System; using System.IO; class Program { static void Main() { string inputPath = @"C:\Users\YourName\Desktop\input.txt"; string outputPath = @"C:\Users\YourName\Desktop\even.txt"; StreamReader reader = new StreamReader(inputPath); StreamWriter writer = new StreamWriter(outputPath); while (!reader.EndOfStream) { int num = int.Parse(reader.ReadLine()); if (num % 2 == 0) { writer.WriteLine(num); } } reader.Close(); writer.Close(); Console.WriteLine("Четните числа са записани в: " + outputPath); } }

трудно Статистика и запис
Файлът grades.txt съдържа само числа 3–6. Намери: брой, сума, средна оценка, брой слаби (<4). Запиши резултатите в stats.txt.
Използвайте пълни пътища и за двата файла.
  • Отвори StreamReader + StreamWriter с пълни пътища
  • Натрупвай sum, count, weak
  • Запиши статистиката форматирано в stats.txt
  • Потвърди на екрана с Console.WriteLine
StreamReader r = new StreamReader(@"C:\Users\...\grades.txt"); StreamWriter w = new StreamWriter(@"C:\Users\...\stats.txt"); int sum=0, count=0, weak=0; while (!r.EndOfStream) { int g = int.Parse(r.ReadLine()); sum += g; count++; if (g < 4) weak++; } r.Close(); w.WriteLine("Брой: " + count); w.WriteLine($"Средна: {(double)sum/count:F2}"); w.WriteLine("Слаби: " + weak); w.Close();
using System; using System.IO; class Program { static void Main() { string gradesPath = @"C:\Users\YourName\Desktop\grades.txt"; string statsPath = @"C:\Users\YourName\Desktop\stats.txt"; StreamReader reader = new StreamReader(gradesPath); int sum = 0, count = 0, weak = 0; while (!reader.EndOfStream) { int grade = int.Parse(reader.ReadLine()); sum += grade; count++; if (grade < 4) weak++; } reader.Close(); StreamWriter writer = new StreamWriter(statsPath); writer.WriteLine("Брой оценки: " + count); writer.WriteLine("Сума: " + sum); writer.WriteLine($"Средна оценка: {(double)sum / count:F2}"); writer.WriteLine("Брой слаби (под 4): " + weak); writer.Close(); Console.WriteLine("Статистиката е записана в: " + statsPath); } }
★
бонус Дневник (append режим)
Напишете програма — "дневник". При всяко стартиране потребителят въвежда бележка, която се добавя в края на файл diary.txt (без да изтрива предишните).
Използвайте пълен път — файлът ще се създаде автоматично при първото стартиране.
  • Използвай StreamWriter с true (append) и пълен път
  • Поискай бележката от потребителя
  • Запиши: [Запис #N]: бележката
  • Бонус: Прочети с StreamReader колко записа вече има и покажи броя
string path = @"C:\Users\...\diary.txt"; int count = 0; if (File.Exists(path)) { StreamReader r = new StreamReader(path); while (!r.EndOfStream) { r.ReadLine(); count++; } r.Close(); } StreamWriter w = new StreamWriter(path, true); Console.Write("Бележка: "); string note = Console.ReadLine(); w.WriteLine($"[Запис #{count+1}]: " + note); w.Close();
using System; using System.IO; class Program { static void Main() { string path = @"C:\Users\YourName\Desktop\diary.txt"; // Първо: брой съществуващи записи int count = 0; if (File.Exists(path)) { StreamReader reader = new StreamReader(path); while (!reader.EndOfStream) { reader.ReadLine(); count++; } reader.Close(); } Console.WriteLine("Дневникът има " + count + " записа."); Console.Write("Нова бележка: "); string note = Console.ReadLine(); // Добавяне (append = true) StreamWriter writer = new StreamWriter(path, true); writer.WriteLine($"[Запис #{count + 1}]: {note}"); writer.Close(); Console.WriteLine("Записано в: " + path); } }
